Команды для навигации и базовой работы с файлами

  • ls — показывает содержимое директории, в которой вы находитесь. Если после команды ввести адрес конкретной директории, то выведет то, что хранится в данной директории. При добавлении аргумента -R также отображаются файлы из поддиректорий.
  • cd — сменить директорию. В качестве параметра принимает адрес директории (абсолютный или относительный). Параметр - возвращает в предыдущую директорию, без параметра команда перемещает в домашнюю директорию.
  • pwd — показывает полный путь до текущей директории.
  • file — показывает тип файла и дополнительную информацию о нем.
  • mv — перемещает файл(ы) в указанную директорию. Эту же команду используют, чтобы поменять имя файла. Для этого не нужно указывать директорию для перемещения, а просто ввести новое название для документа.
  • rm — удаляет директории и их содержимое. rmdir директории — удаляет пустую директорию. rm -R директории — удаляет директорию со всем ее содержимым. rm название_файла — удаляет файл без подтверждения. rm -i название_файла — удаляет файл с запросом на подтверждение (понадобится ввести y для продолжения или n — для отмены).
  • mkdir — создает новую директорию в существующей директории. mkdir -p — дополнительно создает промежуточные родительские директории, если это необходимо.
  • cat — показывает текстовое содержимое документов.
